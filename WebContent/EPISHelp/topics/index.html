Create New PFID

PF ID [New] screen allows you to enter new details related to family, personal and pension nominee. You to add an employee details. When you click on Create new PFID, the following screen opens

 

 

Perform the following tasks to add an employee details:

  • Select type of airport code from Region list box
  • Select type of airport code from Airport Code list box
  • Enter serial number of the airport in Airport Serial Number field.
  • Enter employee code in Employee Code field.
  • Enter name of the employee in Employee Name field.
  • Click on the date picker next to Date of Birth field and select or enter the date of birth
  • Select gender from Sex list box.
  • Select marital status as Yes/No from Marital Status list box .
  • Enter name  of father or husband in Father's / Husband's Name field
  • Select Designation from list box
  • Select the department from the Discipline list box
  • Click on the date picker next to Date of Joining field and select or enter the date of joining
  • Click on the date picker next to Date of Super Annuation field and select or enter the date of super annuation
  • Click on the date picker next to Date of Separation field and select or enter the date of separation
  • Select the reason of separation from Separation Reason list box
  • Enter Permanent Address in Permanent Address field.
  • Enter temporary address in Temporary Address field
  • Enter e-mail ID in Email Id field.
  • Select division from Division list box.
  • Note: NAD stands for local airports and IAD for international airports
  • Select Pension Option Received from Pension Option Received list box.
  • Select Yes/No from Whether Form2 Nomination Received list box to indicate if the form2 was received or not

Note: Selecting Yes displays Nomination for PF section.

.

  • If any remarks, then enter in Remarks field

Family Details:

    • Enter name of the nominee in Name field
    • Click on the date picker next to Date of Birth field and select or enter the date of birth
    • Select relationship of the employee with the employee from Relation with member list box.

Nomination for PF:

    • Select Equal Share Check checkbox to display equal amount f share to be divided among the nominees. If Equal Share Check box is unchecked, then enter manually the share of the nominee in Total Share pay in % field.
    • Enter name of the nominee in Name field
    • Enter nominee’s address in Address field
    • Click on the date picker next to Date of Birth field and select or enter nominee’s date of birth         
    • Select relationship of the employee with the nominee from Relation with member list box.
    • If the age of the nominee is less than 18 years then enter name of the guardian in Name of Guardian field
    • Enter address of the guardian in Address of Guardian field.
    • Based on checking of Equal Share Check field; Total Share pay in % is auto-displayed
  • Click    to add the record.
  • Click  button to clear the data in the fields
  • Click  button to exit from the screen

 

 After entering all information ,when you click on Add first it will show a popup window with an employee list,  if same employee Name  exist in all over India.

 

If you click on OK. It will proceed to open next popup window with an employee list with same date of birth.

 

if you click on ok employee information will add success.
