Search For PF ID

 

PF ID [Search] screen allows you to search details based on Region/Airport code/Employee Name/PF ID/Date of Birth.  When you click on Search for PFID, the following screen opens.

 

To search for employee details, perform all or any of the following tasks:

  • Select region from Region list box
  • Select code of the airport from Airport Code list box
  • Enter name of the employee in Employee Name field
  • Enter central provident fund account number in PF ID field.
  • Select Date of Birth in dd-mm-yyyy Format.
  • Click  button to view employee details’ results
  • Click  button to clear the data in the fields
  • Click  button to exit from the screen

Clicking Search lists all the  employee details.

 

 

 

The information is displayed in columns:

§         PF ID : Displays the central provident fund account number

§         CPF ACC. No: Displays the old provident fund account number

§         Employee Code: Displays code of the employee

§         Employee Name: Displays name of the employee

§         Designation: Displays designation of the employee

§         D.O.B: Displays date of birth of the employee

§         Pension Option: Displays pension option

§         Division: Displays division of the employee

§         Airport Code: Displays code of the airport

§         Region: Displays region of an employee.

§         Check the box below  edit icon to edit the details, a validation message populates asking for your confirmation, opt for Yes or No as per the requisite.
